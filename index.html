Privacy Policy

Perspective Library — Content Distribution Tool
Last updated: July 2026

This tool is a personal automation script built and operated by a single individual creator (Samarth Kulkarni, "Perspective Library") for the sole purpose of publishing the creator's own original video content to the creator's own social media accounts, including a Pinterest Business account.

Who uses this app

This application is used exclusively by its developer, for the developer's own accounts. It is not distributed to, or used by, any third-party users. It does not collect, process, or store data belonging to anyone other than the developer.

What data is accessed

This app requests access to the Pinterest API in order to:

  • Authenticate the developer's own Pinterest Business account via OAuth
  • Create Pins on boards owned by the developer's account
  • Read basic account and board information needed to publish those Pins

No data from any other Pinterest user, board, or account is accessed, collected, or stored.

How data is stored

OAuth access tokens are stored securely using Google Colab's built-in encrypted secrets manager, accessible only to the developer's own environment. No data is shared with, sold to, or processed by any third party.

Data retention

Access tokens are retained only for as long as needed to operate the developer's own publishing workflow, and can be revoked at any time by the developer through Pinterest's account settings.
